About Project CRESSIDA

CRESSIDA is a static prototype of a Polymarket-inspired prediction market platform, developed as a portfolio project for the Human-Computer Interaction module (HCIN6222) at The Independent Institute of Education. This project showcases the application of front-end web development technologies and UI/UX design principles.

How This Website Was Made:

This website was built entirely using client-side technologies:

  • HTML5: For structuring the content.
  • CSS3: For styling, layout (including Flexbox/Grid), and responsive design. Custom CSS variables are used for theme management (dark/light modes).
  • JavaScript (ES6+): For all interactivity, including:
    • Dynamic content rendering (market listings, details, leaderboard).
    • Simulated user registration and login using browser cookies.
    • Simulated trading and balance updates (data is not persisted beyond the session).
    • Client-side form validation.
    • Theme toggling.
  • Chart.js: For displaying historical price trend charts on market detail pages.
  • Font Awesome: For icons.

All data (markets, users, leaderboard entries) is hardcoded within JavaScript files. No backend database or server-side processing is used, making this a fully static website designed to demonstrate front-end capabilities and user interface design. Development occurred around May 2025.
